# StyleMix 👗

App móvil para catalogar la ropa y generar outfits variados sin repetir.

## Stack
- **Backend:** Python + FastAPI
- **Frontend:** React Native (Expo)
- **Base de datos:** PostgreSQL (AWS RDS)
- **Imágenes:** Amazon S3
- **Infra:** AWS

## MVP
- Catalogar prendas con foto
- Etiquetado por categoría, color y ocasión
- Galería del armario con filtros
- Generador de outfits con reglas de no repetición
- Registro de uso
